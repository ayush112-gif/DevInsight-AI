# 🚀 DevInsight AI

> AI-Powered GitHub Repository Analysis Platform

DevInsight AI helps developers understand unfamiliar codebases in minutes instead of hours by automatically analyzing GitHub repositories, detecting technologies, generating architecture insights, and creating AI-powered project summaries.

---

## 📌 Problem Statement

Understanding a new codebase is one of the biggest challenges for developers.

Before contributing to a project, developers often spend significant time figuring out:

* Which technologies are used?
* How is the project structured?
* What are the important files?
* How do different components interact?
* Where should they start?

DevInsight AI solves this by providing an instant, AI-generated understanding of any GitHub repository.

---

## 🖥️ How It Works

1. Paste a GitHub Repository URL
2. DevInsight AI fetches repository metadata
3. AI analyzes project files and dependencies
4. Technologies are detected automatically
5. Architecture insights are generated
6. Results are displayed in an interactive dashboard
